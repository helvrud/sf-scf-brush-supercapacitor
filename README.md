# SF–SCF Modeling of Electroconductive Polymer Brushes

This repository contains the full reproducible workflow for the manuscript:

**O. Rud, S. Chertopalov, O. Borisov,  
"Electroconductive Polymer Brushes for Supercapacitors: Insights from Self-Consistent Field Modeling" (2025)**

## Contents

- `notebook.ipynb` — Jupyter notebook reproducing all figures and analyses
- `cap.py` — Python module implementing the model as python class 
- `run.py` — Python script containing routines needed in notebook.ipynb for running and plotting 
- `sfbox` - a binary executable implementing sf-scf calculations
- `figures/` — Figures generated by the notebook
- `data/` — Optional stored profiles for fast plotting
- `requirements.txt` — file containing all the necessary dependencies fir python virtual environment 

## How to run

```bash
python -m venv venv
pip install -r requirements.txt
. venv/bin/activate
jupyter lab notebook.ipynb
```
